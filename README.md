# Beanora Classic Coffee Website

Elegant, responsive React/Vite coffee-shop website with separate pages, product detail routes, cart and checkout UI.

## Run

```bash
npm install
npm run dev
```

## Pages

- `/` Home
- `/about` About
- `/products` Products
- `/products/:slug` Product detail
- `/gallery` Gallery
- `/cart` Cart
- `/checkout` Checkout
